# Docker develop env

Provide CUDA-10 deep learning and normal development environment.

## Supported env

- python3.6
- cuda10
- pytorch1.0
- tensorflow-nightly
- mxnet-cu10

## Getting start

```bash
# To create the docker image
./docker/deep_start.sh

# Enter docker env
./docker/deep_into.sh
```
